聲控遊戲 
===========
概念發想
-----
用語音辨識操作遊戲，並用喇叭輸出

功能
-----
* 語音辨識
* 玩遊戲
* 喇叭輸出

使用設備
-------
* 樹梅派(pi4) *1
* 麥克風  *1以上
* 喇叭 *1

軟體安裝
-----
* Python3
  * pygame
  * pyaudio
